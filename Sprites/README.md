Carpeta que contiene sprites de Pokémon descargados desde PokeAPI.

## Fuente

Todos los sprites han sido descargados de:
- **PokeAPI**: https://pokeapi.co/
- **GitHub Oficial**: https://github.com/PokeAPI/sprites

## Créditos

Los sprites pertenecen a The Pokémon Company / Nintendo.
Se utilizan únicamente con fines educativos y de demostración.

## Cómo Agregar Más Sprites

Para descargar más Pokémon, usa el patrón de URL:
```
https://raw.githubusercontent.com/PokeAPI/sprites/master/sprites/pokemon/other/official-artwork/{POKEMON_ID}.png
```

Reemplaza `{POKEMON_ID}` con el número Pokédex del Pokémon que quieras descargar.

Ejemplos:
- 1 = Bulbasaur
- 4 = Charmander
- 7 = Squirtle
- 25 = Pikachu
- 149 = Dragonite
